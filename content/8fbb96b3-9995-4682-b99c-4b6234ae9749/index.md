# SQL에서의 데이터 제거하는 방법


## 물리적 삭제


실제로 데이터를 삭제하는 방법이다.


해당 레코드와 관련된 모든 참조도 삭제된다.


따라서 데이터를 복구할 수 없고, 데이터를 완전히 삭제해야 할 때 사용된다.


## 논리적 삭제


실제로 데이터를 삭제하지 않는 방법이다.


플래그나 마크를 추가하여 해당 레코드가 더 이상 사용되지 않거나 유효하지 않는 것을 나타낸다.


다른 레코드에서 해당 레코드를 참조하고 있을 때 유용하다.


논리적 삭제를 사용할 때는 별도의 열이나 속성을 추가해야한다.
